# Revisión pendiente

Esta carpeta contiene material que no forma parte de la estructura principal del repositorio.

## Contenido académico UOC

Los documentos de lógica y fundamentos de computadores se han movido a `contenido-academico-uoc` porque mantienen un enfoque académico y no encajan directamente con la orientación actual del repositorio como base profesional de manuales técnicos.

## Duplicados

Los duplicados detectados se conservan temporalmente en `duplicados` para permitir una revisión manual antes de eliminarlos definitivamente.

## Criterio

El contenido técnico general se ha conservado cuando puede funcionar como manual independiente. El contenido con enfoque universitario, de asignatura o poco alineado con la nueva navegación queda fuera de la estructura principal.

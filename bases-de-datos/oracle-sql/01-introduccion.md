# Oracle SQL

Oracle SQL es la variante de SQL usada en Oracle Database. Mantiene fundamentos relacionales y anade objetos y herramientas propias como sequences, synonyms, packages, PL/SQL, tablespaces y un optimizador muy potente.

## Entorno de practica

Opciones:

- Oracle Database local o en contenedor.
- Oracle SQL Developer.
- SQLcl.
- Oracle Live SQL.

Conexion generica:

```bash
sqlplus usuario/password@host:puerto/servicio
```

## Primeras consultas

```sql
SELECT SYSDATE AS fecha_actual
FROM dual;

CREATE TABLE clientes (
  id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nombre VARCHAR2(100) NOT NULL,
  email VARCHAR2(150) UNIQUE,
  fecha_alta DATE DEFAULT SYSDATE
);

INSERT INTO clientes (nombre, email)
VALUES ('Ana', 'ana@example.com');

COMMIT;
```

## Cuando usar Oracle

- Entornos empresariales.
- Sistemas criticos con alta exigencia operativa.
- Aplicaciones que ya viven en ecosistema Oracle.
- Bases con mucho PL/SQL y tooling corporativo.
- Workloads que necesitan opciones avanzadas de seguridad, particionado y administracion.
